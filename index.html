Get TUN-EYE

Identify and combat the spread of fake news.

Features

Multi-modal input

Accepts both text and image input from the user, allowing for comprehensive fake news detection

Support various browsers

The TUN-EYE extension is available for the most widely used browsers, including: Chrome, Chromium, MS Edge, Opera, Firefox.

Real-time Feedback

The extension displays a result, indicating whether the input content is classified as fake or real.

Multilingual

The trained machine learning model supports the following languages: English, Taglish, Filipino.
